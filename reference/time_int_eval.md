# Resolution of Integral with Respect to Time

Function for the resolution of an integral with respect to time, in the
evaluation of the log-likelihood function. It is implemented as defined
in the paper of Wintrebert et al.'s (2004)

## Usage

``` r
time_int_eval(time_t, k, time_axis)
```

## Arguments

- time_t:

  Event time instant

- k:

  k-th interval of the time-axis

- time_axis:

  Temporal domain

## Value

Evaluation of the temporal integral
