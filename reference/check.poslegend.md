# Check Correctness of Legend Position

The function controls that the provided position of the legend is
correct. It can be either a vector of length 2, giving the x and y
coordinates, or a string, giving the exact position among different
possibilities.

## Usage

``` r
check.poslegend(pos_legend)
```

## Arguments

- pos_legend:

  Either a numerical vector of length 2, with the x and y coordinates,
  or a string with the exact position.

## Value

An error if any condition is not satisfied.
