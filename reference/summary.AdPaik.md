# Summary Method for AdPaik Objects

`summary` method for objects of class `"AdPaik"`. It prepares a
structured summary of the results.

## Usage

``` r
# S3 method for class 'AdPaik'
summary(object, ...)
```

## Arguments

- object:

  An object of class `"AdPaik"`, returned by the main model function.

- ...:

  Additional arguments (currently unused).

## Value

An object of class `"summary.AdPaik"` containing structured model
summary information.
